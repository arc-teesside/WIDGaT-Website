Date picker

Description

A date picker which can also be used as a simple calendar. It can either be displayed directly on the page or attached to a textbox (see attributes for more details).

Screenshots

Attributes

Attach to a textbox Boolean Default: false
If set to true, it will display a textbox and when selected will show the calendar. The selected date will then appear in the textbox and the calendar disappear. This does not affect how the component's output ('Selected date') will be handled.
Date format String Default: standard
This attributes set the date display format. The possible values are 'standard' (dd-mm-yy) or 'us' (mm-dd-yy).
Display multiple months Boolean Default: false
If set to true, the component will display two additional calender showing the months surroind the current month
Display weeks number Boolean Default: fasle
If set to true, the calendar will show the week's number.
Display years/months as menus Boolean Default: false
If set to true, the calendar will show the months and years as menus for rapid selection.
Selected date Date output
The selected date in the calendar. It's a read-only value that can be used as an input to other components.
