Visual countdown

Description

A visual countdown that display a disapearing clock. This component uses HTML5 Canvas

Attributes

Alarm play time Number Default: 3
Defines how long the alarm will ring in seconds. Requires 'Play alarm' to be true.
Mask opacity Number Default: 0.8
Defines the opacity of the mask gradually hidding the clock. Values between 0 and 1 (e.g.:0.1, 0.3,...)
Play alarm Boolean Default: true
Defines wether or not to play an alarm at the end of the countdown
Start time required Number input Default: 2
This defines the time to start counting down from, it is expressed in minutes. This can either be a number or linked to an output (e.g.: textbox, slider,...)
